Projetos

Esse projeto foi desenvolvido com o propósito de treinar e aprofundar meus conhecimentos em HTML, CSS e JavaScript. Nele eu faço um clone da página HOME do site oficial da Vans Brasil. Nesse trabalho eu busco seguir fielmente o layout da página original, respeitando todas as margens e espaçamentos. Os elementos gráficos da página como botões, links, inputs e carrosséis são totalmente funcionais, porém é importante ressaltar que o projeto nesse momento é apenas da página HOME, não possibilitando a navegação para outras páginas do site. O próximo passo desse projeto será aplicar responsividade também para celulares.
