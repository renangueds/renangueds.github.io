Projetos

Esse projeto foi desenvolvido com o propósito de treinar e aprofundar meus conhecimentos em HTML, CSS e JavaScript. Nele eu faço um clone da página HOME do site oficial da Vans Brasil. Nesse trabalho eu busco seguir fielmente o layout da página original, respeitando todas as margens e espaçamentos. Os elementos gráficos da página como botões, links, inputs e carrosséis são totalmente funcionais, porém é importante ressaltar que o projeto nesse momento é apenas da página HOME, não possibilitando a navegação para outras páginas do site. O próximo passo desse projeto será aplicar responsividade também para celulares.

Esse projeto foi elaborado durante conteúdo prático Mission: Origin da Rocketseat. Nesse projeto foi elaborado um site para um salão de beleza e omo desafio foi pedido para deixar o site responsivo. Além do desafio proposto eu criei um código no JS para mudar a cor do botão back-to-top para que ele não fique da mesma cor do footer, assim ele muda para uma cor clara dando mais visibilidade ao botão. Foram feitas também modificações no layout da seção Serviços para o site se adaptar melhor aos tablets. Procurei nesse projeto manter um código organizado e comentado para melhor compreensão de cada passo seguido no projeto.
