Projetos

Home page desenvolvida a partir do layout do site oficial da Vans Brasil. O projeto segue fielmente o design e interações do site original. Click e acesse o projeto online, o código esta disponível em meu github.

Esse projeto foi desenvolvido durante conteúdo prático Mission: Origin da Rocketsea, foi elaborado um site de um salão de beleza. Além de deixar o site responsivo, criei um código em javascript para mudar a cor do botão BACK-TO-TOP para que ele não fique da mesma cor do footer. Foram feitas também modificações no layout da seção Serviços para o site se adaptar melhor aos tablets.

Projeto desenvolvido para uma loja de aplicativos usando display grid, grid-template-row, grid-template-columun e grid-template-areas.
